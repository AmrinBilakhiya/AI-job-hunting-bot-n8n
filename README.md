# 🤖 AI Job Hunting Bot using n8n

## 🚀 Overview
An automated job search system that finds relevant jobs, evaluates them using AI, and generates personalized cover letters.

## ⚙️ Features
- Daily LinkedIn job search automation
- AI-based resume-job matching
- Auto-generated cover letters
- Google Sheets job tracking
- Telegram alerts for high-match jobs

## 🛠 Tech Stack
- n8n (workflow automation)
- Google Sheets & Drive
- Google Gemini API
- Telegram Bot API
